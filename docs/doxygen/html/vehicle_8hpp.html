Detailed Description

Header file for the Vehicle class and its derived classes.

Vehicle class represents an eVTOL vehicle with properties such as cruise speed, battery capacity, energy usage, and fault probability. It includes methods for managing the vehicle's state, such as starting flights, flying, charging, and faults. Derived classes represent specific manufacturers' vehicles.
